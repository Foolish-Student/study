## 题目
给定一个字符串 `colors` 表示一串气球的颜色，以及一个数组 `neededTime` 表示移除每个气球所需的时间。要求移除一些气球，使得**相邻气球颜色不同**，同时总的移除时间最小。

---

## 🧩 思路分析

1. **核心问题**

   * 当出现相邻气球颜色相同的情况时，必须删除其中一些气球；
   * 为了最小化总时间，应尽量**保留移除代价最大的气球**，删除其他气球。

   举例：

   ```
   colors = "aabaa"
   neededTime = [1,2,3,4,1]
   ```

   出现相邻的相同颜色段 “aa”：

   * 第一个 “aa”：删除代价较小的（1），保留较大的（2）；
   * 第二个 “aa”：删除代价较小的（1），保留较大的（4）；
     总时间 = 1 + 1 = 2。

---

2. **按连续相同颜色分段处理**

   * 从左到右扫描字符串；
   * 用两个变量：

     * `sum` 记录当前连续同色段所有移除时间之和；
     * `mx` 记录该段中最大的移除时间；
   * 每当颜色变化时（即遇到新颜色）：

     * 当前段的最小移除代价 = `sum - mx`；
     * 加入总结果；
     * 重新开始下一段。

   最后再加上最后一段的结果。

---

3. **贪心思想说明**

   * 对于每一段连续相同颜色的气球，只需要删除除去一个（保留代价最大的）；
   * 因此，只需计算该段内“要删的总时间 = 总和 - 最大值”；
   * 遍历所有颜色段即可得到全局最优解。

---

## 🧮 示例分析

### 示例 1：

```
colors = "abaac"
neededTime = [1,2,3,4,5]
```

* “a”(0) → “b”(1)：不同色；
* “b”(1) → “a”(2)：不同色；
* “a”(2) → “a”(3)：同色段 → 删除较小的 3；
* “a”(3) → “c”(4)：不同色；
  ✅ 总最短时间 = 3。

---

### 示例 2：

```
colors = "abc"
neededTime = [1,2,3]
```

没有连续相同颜色 → 无需删除；
✅ 输出 0。

---

### 示例 3：

```
colors = "aabaa"
neededTime = [1,2,3,4,1]
```

相同颜色段：

* “aa”(0,1)：删除 1；
* “aa”(3,4)：删除 1；
  ✅ 总最短时间 = 2。

---

## 🧠 复杂度分析

* **时间复杂度：** O(n)
  只需遍历一次字符串；
* **空间复杂度：** O(1)
  仅使用常数辅助变量。

---

## ✅ 总结

这是一道经典的**贪心算法 + 分段累加**问题。
关键在于：

* 对连续相同颜色分段处理；
* 保留代价最大的一个；
* 删除其余，累加总时间。

整个算法逻辑简洁、效率高，非常适合大规模输入场景。
