# 3217.从链表中移除在数组中存在的节点
## 题目
给定一个整数数组 `nums` 和一个链表的头节点 `head`。
任务是：**从链表中移除所有节点值存在于 `nums` 中的节点**，并返回修改后的链表头节点。

---

## 🧩 思路分析

1. **问题本质**

   * 题目要求删除所有节点值属于 `nums` 的节点；
   * 这是典型的“链表节点删除”问题，只不过删除条件由一个数组提供；
   * 因此，关键在于高效判断“某个节点值是否存在于 `nums` 中”。

2. **高效查找：使用哈希表**

   * 如果直接在 `nums` 中查找，会导致 O(n × m) 的复杂度；
   * 因此，我们可以先把 `nums` 中的所有值存入一个哈希表（或 `unordered_map` / `unordered_set`）；
   * 查找操作可降为 O(1)。

   举例：

   ```
   nums = [1,2,3]
   建立哈希表 isExist：
   {1: true, 2: true, 3: true}
   ```

---

3. **删除链表节点的标准技巧**

   * 链表删除节点时，需要特别处理头节点；
   * 因此使用一个**哑节点（哨兵节点，dummy/sentry）**来统一逻辑；
   * 设一个指针 `p` 指向当前节点的前驱；
   * 当 `p->next->val` 在哈希表中时，说明该节点需要删除：

     ```
     p->next = p->next->next
     ```

     否则，移动指针：

     ```
     p = p->next
     ```
   * 这样可以保证在 O(n) 时间内遍历并修改整个链表。

---

4. **完整的处理逻辑**

   * 第一步：把 `nums` 中的值存入哈希表；
   * 第二步：使用哑节点连接原链表；
   * 第三步：遍历链表，遇到需要删除的节点就跳过；
   * 第四步：返回哑节点的下一个节点（新的头节点）。

---

## 🧮 示例解析

### 示例 1

```
nums = [1,2,3]
head = [1,2,3,4,5]
```

* 哈希表：{1, 2, 3}
* 遍历：

  * 节点 1 → 删除；
  * 节点 2 → 删除；
  * 节点 3 → 删除；
  * 剩余 [4, 5]
    ✅ 结果：[4,5]

---

### 示例 2

```
nums = [1]
head = [1,2,1,2,1,2]
```

* 哈希表：{1}
* 遍历：

  * 删除所有值为 1 的节点；
  * 剩余 [2,2,2]
    ✅ 结果：[2,2,2]

---

### 示例 3

```
nums = [5]
head = [1,2,3,4]
```

* 哈希表：{5}
* 没有匹配项；
  ✅ 结果：[1,2,3,4]

---

## 🧠 复杂度分析

* **时间复杂度**：
  O(n + m)，其中 n 为链表长度，m 为数组长度。

  * 建立哈希表 O(m)
  * 遍历链表 O(n)

* **空间复杂度**：
  O(m)，用于哈希表存储 `nums` 中的元素。

---

## ✅ 总结

这是一道**链表节点删除 + 哈希查找**的综合题。
核心技巧包括：

1. 使用哈希表加速查找；
2. 使用哑节点简化删除逻辑；
3. 一次遍历完成所有删除操作。

整体结构简洁清晰，时间效率优越。
