# 3289. 数字小镇中的捣蛋鬼

## 题目描述

数字小镇 Digitville 中有一个整数列表 `nums`，包含从 `0` 到 `n - 1` 的数字。每个数字**本应只出现一次**，但其中有 **两个** 顽皮的数字**额外多出现了一次**，使列表长度比正常情况更长。

请找出这两个重复出现的数字，并返回一个长度为 2 的数组（顺序任意）。

---

## 示例

**示例 1**
输入：`nums = [0,1,1,0]`
输出：`[0,1]`
解释：数字 0 和 1 在数组中各出现了两次。

**示例 2**
输入：`nums = [0,3,2,1,3,2]`
输出：`[2,3]`
解释：数字 2 和 3 在数组中各出现了两次。

**示例 3**
输入：`nums = [7,1,5,4,3,4,6,0,9,5,8,2]`
输出：`[4,5]`
解释：数字 4 和 5 在数组中各出现了两次。

---

## 提示

* `2 <= n <= 100`
* `nums.length == n + 2`
* `0 <= nums[i] < n`
* 输入保证 `nums` 中**恰好**包含两个重复的元素。
